# Push to GitHub

Your repo is initialized and committed. To create the GitHub repo and push:

## 1. Create the repo on GitHub

1. Go to **https://github.com/new**
2. Repository name: `manim-testing` (or `Manim-testing`)
3. Description: `Manim renderer for Vynotes with plugins`
4. Choose **Public**
5. **Do NOT** initialize with README, .gitignore, or license (we already have them)
6. Click **Create repository**

## 2. Push your code

Run these commands (replace `YOUR_USERNAME` with your GitHub username):

```powershell
cd "c:\Users\Hp\PycharmProjects\Manim testing"

git remote add origin https://github.com/YOUR_USERNAME/manim-testing.git
git push -u origin main
```

If you use **IshanKalra007**:

```powershell
git remote add origin https://github.com/IshanKalra007/manim-testing.git
git push -u origin main
```

## 3. Done

Your repo will be live at `https://github.com/YOUR_USERNAME/manim-testing`
